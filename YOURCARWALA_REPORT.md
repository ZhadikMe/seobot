## Что конкретно изменилось

### Canonical URL
**До:** `rel="canonical" href="index.html"` — относительный, бесполезен для поисковиков

**После:** `rel="canonical" href="https://yourcarwala.com/a-quick-guide-to-engine-bearing-productivity/"` — абсолютный с реальным доменом

---

### Hreflang
**До:** отсутствовал полностью

**После:** на каждой статье добавлен блок из 14 тегов:
```html
<link rel="alternate" hreflang="en"  href="https://yourcarwala.com/a-quick-guide-.../">
<link rel="alternate" hreflang="cs"  href="https://yourcarwala.com/cs/a-quick-guide-.../">
<link rel="alternate" hreflang="de"  href="https://yourcarwala.com/de/a-quick-guide-.../">
<link rel="alternate" hreflang="es"  href="https://yourcarwala.com/es/a-quick-guide-.../">
...и т.д. для  12 языков + x-default
```

---

### OG Image
**До:** `og:image` отсутствовал на 10 страницах (поделиться в соцсетях — без превью)

**После:** `og:image` добавлен из первого изображения страницы:
```html
<meta property="og:image" content="/wp-content/uploads/2022/09/image1-4.jpg">
```

---

### Nofollow
**До:** часть внешних ссылок без атрибута `rel` — передавали вес чужим сайтам

**После:** все внешние ссылки:
```html
<a href="https://external.com" rel="nofollow noopener noreferrer">...</a>
```

## Что не входит в возможности без хостинга

Следующие SEO-улучшения **технически возможны**, но требуют живого задеплоенного сайта:

| Функция | Почему недоступно | Что нужно |
|---|---|---|
| 301 редиректы | Нет веб-сервера для конфигурации | Nginx / .htaccess на хостинге |
| Google Indexing API | Нет публичного URL для верификации | Верифицированный домен в GSC |
| Google Search Console | Нет доступа к домену | DNS-верификация или файл в корне |
| Core Web Vitals (LCP, CLS) | Нет данных без реального трафика | Хостинг + реальные посетители |
| Автоиндексация через ping | Нет живого URL | Хостинг |
| HTTPS (SSL) | — | Хостинг с SSL-сертификатом |

**Всё перечисленное выше готово к внедрению** — мета-теги, canonical, hreflang уже прописаны корректно и заработают сразу после деплоя.
